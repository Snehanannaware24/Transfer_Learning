# Transfer_Learning

# Description
This project implements transfer learning using ResNet50 for brain tumor classification.

# Dataset
Brain MRI Dataset: https://www.kaggle.com/datasets/navoneel/brain-mri-images-for-brain-tumor-detection

# Steps 
Data preprocessing and augmentation
Model training using ResNet50
Fine-tuning
Evaluation with metrics

# Tools Used
TensorFlow
Keras
Google Colab
